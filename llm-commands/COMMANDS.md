## 🚀 快速開始

### 在 VS Code 使用

1. 開啟專案資料夾
2. 啟動 LLM（Cascade/Roo Code/Copilot）
3. 在聊天視窗輸入指令

### 指令格式

```
[動作] + [內容] + [細節]
```

**範例**：
```
新增一個活動：聖誕節慶典，12月25日下午3點，地點主堂
```

## ⚠️ 注意事項

### 權限管理

某些操作需要特定權限：

- ✅ **一般同工**：新增活動、講道、代禱
- ✅ **編輯者**：修改內容、審核代禱
- ⚠️ **管理員**：修改樣式、刪除資料、系統設定

### 安全提醒

1. **審核機制**
   - 代禱事項需審核後顯示
   - 見證分享需審核後發布
   - 重要內容變更需通知管理員

2. **敏感資料**
   - 不公開個人電話
   - 不公開個人地址
   - 財務資訊需加密

3. **備份建議**
   - 每週自動備份所有資料
   - 重大變更前手動備份
   - 備份檔案異地保存

## 🆘 常見問題

### Q: 指令輸入錯誤怎麼辦？

A: LLM 會提示錯誤並要求重新輸入，或提供修正建議。

### Q: 可以撤銷操作嗎？

A: 可以，輸入：
```
撤銷上一個操作
```

或回復到特定時間點：
```
回復到 2024-01-01 的版本
```

### Q: 如何查看操作歷史？

A: 輸入：
```
顯示最近10次操作記錄
```

### Q: 批次操作會覆蓋現有資料嗎？

A: 預設為「合併」模式，不會覆蓋。若需覆蓋，請明確指定：
```
匯入資料並覆蓋重複項目
```

## 💡 最佳實踐

### 1. 清晰明確

✅ **好**：
```
把主日崇拜時間改為 10:00-12:00
```

❌ **不好**：
```
改時間
```

### 2. 提供完整資訊

✅ **好**：
```
新增活動：聖誕節慶典
時間：2024-12-25 15:00-18:00
地點：主堂
說明：歡迎攜家帶眷參加
```

❌ **不好**：
```
新增聖誕節活動
```

### 3. 確認重要操作

在執行刪除、覆蓋等操作前，LLM 會要求確認：

```
確定要刪除活動「聖誕節慶典」嗎？(yes/no)
```
